# Chat
基于Laravel + Vue， 使用socket.io实现的移动端实时通信
执行命令
```
npm install
composer install
//修改env文件，配置数据库
php artisan migrate
npm run dev
```
### Complete
- 项目文件初始化，依赖文件配置
- 整合laravel+vue
- socket封装
- 模拟数据
- 1对1聊天
